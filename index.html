One-time setup — takes about 5 minutes
① Host your app on HTTPS (free)
1. Open netlify.com/drop on your PC browser
2. Drag your Attendance App v2 ENHANCED folder into the page
3. Wait 10 seconds — you get a free link like:
  https://random-name.netlify.app
4. Copy that link and paste below 👇
② Get a free Google Client ID
1. Go to console.cloud.google.com
2. Click "Select a project" → "New Project"
  Name: GG Attendance → Create
3. Left menu → APIs & Services → Library
  Search: Google Sheets API → Enable
4. Left menu → Credentials → Create Credentials → OAuth client ID
  Type: Web application
  Under Authorized JavaScript origins → Add your Netlify URL
5. Click Create → Copy the Client ID → paste below 👇
